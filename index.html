Latitud

The purpose of this project was to analyze how weather changes as you get closer to the equator.To accomplish this analysis, frst pulled data from the OpenWeatherMap API to assemble a dataset on over 500 cities.

After assembling the dataset, we used Matplotlib to plot various, aspects of the weather vs latitud.Factors we looked at incluide:

temperature, cloudiness, wind speed and humidity.This provides the source data visualizations created as part of the analysis, as well as explanations and descriptions of any trends and correlations witnessed
